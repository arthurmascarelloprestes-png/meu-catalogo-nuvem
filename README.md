
# 🛒 Meu Catálogo

Este é um sistema de catálogo interativo desenvolvido para facilitar a visualização e gestão de produtos. O projeto foi construído com foco em performance, persistência de dados e uma interface amigável para o usuário.

## 🚀 Funcionalidades
* Exibição dinâmica de produtos.
* Integração com banco de dados em tempo real.
* Interface responsiva (funciona no celular e no computador).

## 🛠 Tecnologias Utilizadas
* **HTML5:** Estrutura semântica do projeto.
* **CSS3:** Estilização e layout (Flexbox/Grid).
* **JavaScript (ES6+):** Lógica do sistema e manipulação do DOM.
* **Supabase:** Backend as a Service (BaaS) para autenticação e banco de dados SQL.
